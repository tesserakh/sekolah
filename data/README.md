# Dataset

## Version 2

Fields:

1. NPSN (school number)
2. Name
3. Level (type of school)
4. Status
5. Address
6. Subdistrict
7. City
8. Province
9. Akreditasi
10. Kepala Sekolah
11. Guru
12. Siswa Laki-laki
13. Siswa Perempuan
14. Rombongan Belajar
15. Kurikulum
16. Semester Data
17. Ruang Kelas
18. Link (to profile page)

Sorting alphabetically by Level, Status, Province, City, then Address

One school per row

Total 555,898 rows
